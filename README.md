# Urban Traffic Analytics

Real-time simulation and 3D map analysis of urban traffic density across Turkey. Built with Python, Streamlit, and PyDeck.

## Features
* **Live 3D Visualization:** Interactive mapping of active vehicles using PyDeck and Carto dark theme.
* **Time-Series Simulation:** 24-hour traffic density modeling based on real datasets.
* **Data Pipeline:** Automated cleaning and processing of raw vehicle statistics.

## Tech Stack
* Python
* Streamlit
* Pandas & NumPy
* PyDeck
